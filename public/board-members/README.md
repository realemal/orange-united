# School Board Member Images

Place board member headshots here.

## File naming:
- `member-1.jpg` - Board Member 1 (Area 1)
- `member-2.jpg` - Board Member 2 (Area 2)
- etc.

## Instructions to get real photos:

1. Visit https://www.orangeusd.org/Board-of-Education/Board-Members/index.html
2. Right-click on each board member photo
3. Save images as `member-1.jpg`, `member-2.jpg`, etc.
4. Place in this folder

Or use placeholder images until you have real ones.
